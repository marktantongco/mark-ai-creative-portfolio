## Design Philosophy

### Three Design Approaches

The portfolio showcases three distinct visual languages, each with a unique animation stack:

1. **Brutalist Industrial** — Raw, honest, unapologetic. GSAP ScrollTrigger + monospace grids. No decoration — only function.
2. **Organic Minimalism** — Calm, breathing, tactile. Framer Motion spring physics + earth tones. Generous whitespace as design choice.
3. **Cyberpunk Neon** — Intense, futuristic, alive. Three.js + R3F + GSAP + neon FX. A hacker control center aesthetic.

## Getting Started

### Prerequisites

- Node.js 18+ or Bun 1.0+
- npm, yarn, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/marktantongco/mark-ai-creative-portfolio.git
cd mark-ai-creative-portfolio

# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build
```

### Environment Variables

Create a `.env.local` file for local development:

```env
# AI SDK (if using API routes)
AI_GATEWAY_API_KEY=your_key_here
```
